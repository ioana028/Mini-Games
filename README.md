# Mini Games (Learning Project)   ☆
You can see it here: https://mini-games028.netlify.app/

This project is a small collection of browser games created while learning **JavaScript, React, and the HTML Canvas API**.

The main goal was to practice building interactive applications, implementing game logic, and improving front-end development skills.

## Games  

###  Memory Game (React)  ☆

A card matching game built with React.

Features:

* Dynamic grid size
* Card flip logic
* Match detection
* Win detection and reset functionality

###  Pacman (Canvas + JavaScript)  ☆

A simplified Pacman-style game built using the HTML Canvas API.
Please note that this project is based on a YouTube tutorial and was recreated for learning purposes.

Features:

* Tile-based map
* Keyboard movement
* Ghost movement logic
* Collision detection
* Score and lives system

## Tech Stack

* React
* JavaScript (ES6)
* HTML5 Canvas
* CSS
* Vite

## Purpose

This project was built as part of the learning process to better understand:

* React component structure
* State management
* Game loops and movement logic
* Collision detection
* Canvas rendering

More games may be added as the learning journey continues.
